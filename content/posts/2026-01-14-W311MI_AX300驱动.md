我在debian12和debian13上试过.

首先先安装一下依赖

```bash
sudo apt update
sudo apt install linux-headers-$(uname -r) build-essential
```

然后打开官网



![alt text](https://cdn.tungchiahui.cn/tungwebsite/assets/images/2026/01/14/1768404784719.webp)

![alt text](https://cdn.tungchiahui.cn/tungwebsite/assets/images/2026/01/14/1768404847119.webp)

![alt text](https://cdn.tungchiahui.cn/tungwebsite/assets/images/2026/01/14/1768404878732.webp)

![alt text](https://cdn.tungchiahui.cn/tungwebsite/assets/images/2026/01/14/1768404891239.webp)

解压出来

![alt text](https://cdn.tungchiahui.cn/tungwebsite/assets/images/2026/01/14/1768404919231.webp)

```bash
sudo dpkg -i ax300-wifi-adapter-linux-driver.deb
```

然后显示`aic8800dc driver install successfully`才是成功安装.
